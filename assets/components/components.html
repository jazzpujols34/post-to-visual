Interactive components

Self-contained vanilla-JS snippets — no dependencies. Each demo below is live; the copy button gives you a standalone block (markup + scoped script) to paste into a page. Toggle the theme to confirm a widget reads in both modes.

The rule: earn it. Don't add a widget by default. Reach for one only when the content genuinely suits it — a self-assessment for "where do you sit" pieces, a slider for redesigns. A bolted-on quiz makes a page worse, not better.
