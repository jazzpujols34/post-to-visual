To apply a palette to a page: open themes.css, copy the chosen palette's LIGHT token block into the page's :root{}, and the DARK block into @media (prefers-color-scheme: dark){ :root{ … } }. The token names are accent slots, so existing markup and svg-recipes keep working — only the hues change. Run verify.py after; the green box above uses --green-solid on purpose, so confirm it stays readable when you flip to dark.
